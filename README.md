# VPN Vless Project

Телеграм бот с автовыдачей VLESS-VPN конфигураций через Remnawave API

## Локальный запуск
Перед запуском необходимо создать свою Remnawave панель!
1) Создать новую БД PostgreSQL
2) Настроить вебхуки (Ngrok, Nginx или аналогичные сервисы)
3) Создать новый .env файл (используя .env.example)
4) Запустить RabbitMQ на локальной машине
6) Запустить первый терминал и прописать `fastapi dev .\src\main.py`
7) Запустить второй терминал и прописать `taskiq scheduler src.client.taskiq:scheduler`


## Запуск в Docker-контейнере
1) Выполнить c 1 по 4 шаги из предыдущего summary
2) Установить docker на свою систему
3) Прописать в консоли `docker compose up`, можно добавить флаг `-d`, если вам нужен терминал
